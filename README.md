
# wilon.github.io

My notes show page.

Fast search, copy.


### Install && Start

Require [gulp.js](http://gulpjs.com/)

    git clone https://github.com/wilon/wilon.github.io.git
    cd wilon.github.io
    npm install && npm start

Then open [http://localhost:3000](http://localhost:3000) in your browser.

### Update Notes

1. Write notes

    `data/SOME_NOTES.md`

2. Add icon

    `blog/images/SOME_NOTES.png`

3. Browser Sync
